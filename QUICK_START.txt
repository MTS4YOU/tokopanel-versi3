🚀 ONE-COMMAND DEPLOY

  VERCEL (Easiest):
    1. git add . && git commit -m "v1.0.0: Production ready"
    2. git push origin main
    3. https://vercel.com → New Project → Import repo
    4. Set env vars (MONGODB_URI, MONGODB_DB_NAME, NODE_ENV)
    5. Click Deploy → Done! ✨

  DOCKER:
    docker build -t tokopanel-v3 .
    docker run -e MONGODB_URI=... -p 3000:3000 tokopanel-v3

  RAILWAY:
    railway login
    railway init
    railway variables set MONGODB_URI=...
    railway up


🔐 ENVIRONMENT VARIABLES (REQUIRED)

  MONGODB_URI=mongodb+srv://user:pass@cluster.mongodb.net/?...
  MONGODB_DB_NAME=tokopanel_admin
  NODE_ENV=production

  (See .env.example for complete template)
